# Employee Management System - Technical Test

This Java console application was developed as a technical assessment. It demonstrates proficiency in core Java concepts, including the **Stream API**, **BigDecimal** for financial precision, and clean code architecture.

## 🚀 How to Run in Eclipse

Follow these steps to import and execute the project:

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/Hendrick201/Pactice-Test

   
2. Import into Eclipse:

Open Eclipse IDE.

Go to File > Import....

Select General > Existing Projects into Workspace and click Next.

Click Browse next to Select root directory and choose the cloned project folder.

Ensure the project is checked and click Finish.

3. Environment Setup:

Ensure you have JDK 19 or higher configured in your Eclipse.

If needed, right-click the project > Build Path > Configure Build Path... to verify your JRE System Library.

4. Run the Application:

Locate Main.java inside the Program package.

Right-click the file and select Run As > Java Application.
```
